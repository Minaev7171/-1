# Профессиональный Discord Бот для модерации и тикетов

Этот бот предназначен для управления сервером Discord с использованием мощной базы данных MongoDB. Он поддерживает несколько серверов, имеет систему логирования, тикетов, модерации и подробную статистику.

## Основные функции

-   **Модерация**: Кик, бан, предупреждения (варны), очистка сообщений.
-   **Система Мутов**: Работает через встроенные тайм-ауты Discord.
-   **Тикеты**: Создание каналов поддержки с помощью кнопок.
-   **Логирование**: Отслеживание удаленных и измененных сообщений, входа и выхода участников.
-   **Статистика**: Просмотр количества действий модерации на сервере.
-   **Настройка через Discord**: Полное управление ботом (включение/выключение модерации, выбор каналов) через интерактивное меню.
-   **Авто-настройка**: Команды для автоматического создания необходимых ролей и каналов.

## Команды

### Модерация
-   `!кик @участник [причина]` — Исключить участника.
-   `!бан @участник [причина]` — Забанить участника.
-   `!мут @участник [минуты] [причина]` — Отправить в тайм-аут.
-   `!варн @участник [причина]` — Выдать предупреждение.
-   `!варны @участник` — Список предупреждений.
-   `!очистить [количество]` — Удалить сообщения.

### Настройка и Утилиты
-   `!настройки` — Панель настроек (кнопки/меню).
-   `!статистика` — Статистика сервера.
-   `!панель_тикетов` — Создать сообщение для открытия тикетов.
-   `!настройка_мод` — Авто-создание ролей и лог-канала.
-   `!настройка_тикетов` — Авто-создание категории для тикетов.

## Установка и запуск

### 1. Требования
- Python 3.8+
- База данных MongoDB (можно использовать бесплатный кластер на [MongoDB Atlas](https://www.mongodb.com/cloud/atlas))

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка переменных окружения
Создайте файл `.env` в корневой папке бота и добавьте туда следующее:
```env
DISCORD_TOKEN=ваш_токен_бота
MONGO_URI=ваша_ссылка_на_mongodb
```

### 4. Запуск
```bash
python main.py
```

## Развертывание на бесплатный хостинг (например, Render или Koyeb)

1.  Создайте репозиторий на GitHub и загрузите туда код.
2.  Зарегистрируйтесь на [Render](https://render.com/) или [Koyeb](https://www.koyeb.com/).
3.  Создайте новый "Web Service" или "Worker".
4.  Подключите свой GitHub репозиторий.
5.  В настройках (Environment Variables) добавьте `DISCORD_TOKEN` и `MONGO_URI`.
6.  Команда запуска: `python main.py`.
7.  Бот будет автоматически запущен и будет работать 24/7 (в зависимости от ограничений бесплатного тарифа).

*Примечание: Для MongoDB Atlas не забудьте добавить IP-адрес хостинга в WhiteList (или разрешите доступ со всех IP 0.0.0.0/0 для теста).*
